The Greek Alphabet

Biblical Greek uses a 24-letter alphabet inherited from classical Greek. The New Testament was written in Koine Greek, the common dialect of the eastern Mediterranean world.

Breathing Marks

Words beginning with vowels or rho receive breathing marks. Rough breathing indicates an h sound.

Mark Name Example
᾿ Smooth Breathing ἀνήρ
῾ Rough Breathing ἁμαρτία

The Greek Cases

Greek nouns change endings according to grammatical function.

Case Function Example
Nominative Subject ὁ λόγος
Genitive Possession τοῦ λόγου
Dative Indirect Object τῷ λόγῳ
Accusative Direct Object τὸν λόγον

Greek Word Order

Greek word order is flexible because grammatical relationships are shown through inflection rather than strict position.

ἐν ἀρχῇ ἦν ὁ λόγος
“In the beginning was the Word.”

Conditional Sentences

Greek conditional sentences use εἰ and ἐάν constructions to express various kinds of conditions.

First Class Condition: Assumes the condition for the sake of argument.
